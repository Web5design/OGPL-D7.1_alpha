INSTALLATION:

1- Make sure that your hosting environment meets the following requirements: http://drupal.org/requirements
2- Download/Clone OGPL7 alpha distribution:
  Option a-ZIP: https://github.com/REI-Systems/opendatagov/archive/master.zip
  Option b-git(make sure that git is installed on your system): git clone https://github.com/REI-Systems/opendatagov.git
3- Follow these instruction(ignore "Download Drupal" section): http://drupal.org/documentation/install/beginners
  *Note: On the profile selection page, choose "OGPL7" instead of standard
4- Enjoy

ALTERNATIVE INSTALLATION 
In case drush 5.8 or higher is installed on your system and drupal virtual enviroment has been set-up, run the following command:

drush si ogpl7 --sites-subdir=<domain> --db-url=mysql://<username>:<password>@<domain>:<port>/<database> --account-name=<username> --account-mail=<accountemail> --account-pass=<userpassword> --site-mail=<siteemail> --site-name=<sitename>
