INSTALLATION:

1- Make sure that your hosting environment meets the following requirements: http://drupal.org/requirements
2- Download/Clone OGPL7 alpha distribution:
  Option a(ZIP): https://github.com/REI-Systems/opendatagov/archive/master.zip
  Option b(GIT, make sure it's installed): git clone https://github.com/REI-Systems/opendatagov.git
3- Follow these instruction(ignore "Download Drupal" section): http://drupal.org/documentation/install/beginners
  *Note: On the profile selection page, choose "OGPL7" instead of "Standard"
4- Enjoy!

ALTERNATIVE INSTALLATION:

1- Make sure that your hosting environment meets the following requirements: http://drupal.org/requirements and both git and drush 5.8 (or later) are installed
2- Run: git clone https://github.com/REI-Systems/opendatagov.git
3- Set-up the hosting environment for the cloned drupal distribution
4- Run: drush si ogpl7 --sites-subdir=<domain> --db-url=mysql://<username>:<password>@<domain>:<port>/<database> --account-name=<username> --account-mail=<accountemail> --account-pass=<userpassword> --site-mail=<siteemail> --site-name=<sitename>
5- Enjoy!
