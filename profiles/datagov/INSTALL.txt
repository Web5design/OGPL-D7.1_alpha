There are two ways to install the Open Datagov distribution:
1- 
  a) Download the distribution or clone it from github
  b) Untar datagov distribution
  c) Set up a virtual server for it (i.e. through MAMP)
  d) Go to /install.php and select "Datagov Distro" installation profile
  e) Follow the instructions
  f) Enable desired features
  g) *Ignore the warning that is going to appear on the screen for now 
  h) Enjoy!


2-
  a) run ./datagov.drush (located in datagov's profile root folder)
  b) make sure that drush is installed on local system and that the path to drush is correct
  with the datagov.drush script
  c) Enjoy!
